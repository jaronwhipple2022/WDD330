"This" in JavaScript

In JavaScript, keyword this behaves unlike most other programming languages. It can be used in any function, even if itâ€™s not a method of an object.
That way incase a variable gets changed somewhere else within the code this will still work and reference the corect variable.
The value of this is evaluated during the run-time, depending on the context.

Chapter 5: objects

An object in JavaScript is a self-contained set of related values and functions.

easiest way to create an object literal:
const superman = {};
ways to call parts of it:
superman.name
superman['name']

check to see if an object contains a property:
'city' in superman; --returns t/f
=> hasOwnProperty()

How to look up values ex.

to get object values: Object.values()
this will return the objects values
for(const value of Object.values(superman)) {
console.log(value);}

to get key value pairs: Object.values()
